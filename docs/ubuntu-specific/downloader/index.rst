.. _downloaders:

===========
Downloaders
===========

This includes how to install Transmission's headless downloading program and how to install OpenVPN's stuff with direct use of `PIA's <privateinternetaccess.com>`_ service and files.

This will also include, in the future, a how-to install `NZBGET <nzbget.net>`_ program. This is used to download, unpack, decompress, and a bunch of other things, Usenet files.
